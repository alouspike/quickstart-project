# quickstart-parcel
Quickstart front project with parcel bundler
# Project structure

```shell
.
├── LICENSE
├── README.md
├── index.html
├── package.json
├── src
│   ├── js
│   │   └── shop.js
│   ├── scss
│   │   ├── _base.scss
│   │   └── _reset.scss
│   ├── index.js
│   ├── style.scss
└── .babelrc

```
# Install dependencies
` npm i`

# Dev
` npm run start `

# watch
` npm run watch `

# build
` npm run build `
Your bundled project is available at `dist` filder.
